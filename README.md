# esp32-marauder
Custom esp32-marauder

# CONNECTION
esp32 -> TFT Touch 240x320px Display 2.8 Inch
- 3.3V -> VCC
- GND -> GND
- CS -> D17
- RESET -> D5
- DC -> D
- MISO -> D19
- MOSI -> D23
- SCK -> D18
- T_SCK -> D18
- T_MOSI -> D23
- T_MISO -> D19

## Builtin SD Card Module (is optional)
- SD_MISO -> D19
- SD_MOSI -> D23
- SD_SCK -> D18
- SD_CS -> D12
